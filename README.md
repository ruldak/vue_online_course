# Online Course Platform - Frontend Showcase

## Introduction

This project is the frontend client for a full-featured online course platform. It is built with Vue.js to serve as a hands-on showcase and interactive demonstration of the robust backend API developed with the Django REST Framework.

The primary focus of the overall project is the backend system. This frontend application was created to consume the API, allowing users to interact with and evaluate the backend's capabilities in a tangible way.

## Backend API Features Demonstrated

This frontend application showcases the following key features of the backend API:

*   **Secure Payment Integration:** Fully functional payment system supporting both **Stripe** and **PayPal** for processing course purchases.
*   **Shopping Cart System:** A complete "Add to Cart" and checkout workflow.
*   **Advanced Filtering:** Implementation of Django Filter for sorting and filtering courses based on multiple criteria, such as:
    *   Category
    *   Price
    *   Ratings
    *   And more.
*   **User Authentication & Authorization**
*   **Course and Lesson Management**

## Technology Stack

*   **Frontend:** Vue.js, Vue Router
*   **Backend:** Django REST Framework, Python

## Getting Started

To run this frontend application locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/ruldak/vue_online_course.git your_directory
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd your_directory
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Run the development server:**
    ```bash
    npm run dev
    ```
